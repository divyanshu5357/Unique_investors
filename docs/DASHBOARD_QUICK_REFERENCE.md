## 🎨 Color System

### Primary Colors
```
🟢 EMERALD  #10B981  → Success, Completed
🔵 BLUE     #0EA5E9  → Primary, Info
🟡 AMBER    #F59E0B  → Pending, Warning
🔴 RED      #EF4444  → Error, Alert
🟣 PURPLE   #A855F7  → Secondary, Accents
🟠 ORANGE   #FF8C00  → Charts, Performance
🔷 INDIGO   #6366F1  → Special, Emphasis
```

### Gradient System
```
Card Backgrounds:
  - from-{color}-50 to-{color}-100       (Light mode)
  - dark:from-{color}-950/30 to-{color}-900/20  (Dark mode)

Header Gradients:
  - from-{color}-500 via-{color}-500 to-{color}-500
  - dark:from-{color}-600 to-{color}-600
```

## 📐 Responsive Breakpoints

```
MOBILE (< 640px)
├─ Single column layout
├─ Full-width cards
├─ Small text (text-xs to text-base)
├─ Stacked components
└─ 100% responsive

TABLET (640px - 1024px)
├─ Two column layout (mostly)
├─ Better spacing
├─ Medium text (text-sm to text-lg)
├─ Balanced grid
└─ Full-width optimized

DESKTOP (1024px+)
├─ Three column layout (where applicable)
├─ Generous spacing
├─ Larger text (text-base to text-xl)
├─ Professional appearance
└─ Centered, constrained width
```

## 🔄 Responsive Sizing Examples

```tsx
// Text
text-2xl sm:text-3xl md:text-4xl

// Padding
p-3 sm:p-4 md:p-6

// Gap/Spacing
gap-3 sm:gap-4 md:gap-6

// Grid
grid-cols-1 sm:grid-cols-2 lg:grid-cols-3

// Icons
h-5 w-5 sm:h-6 sm:w-6
```

---

## 🌙 Dark Mode Support

```tsx
// Light Mode
from-slate-50 via-blue-50 to-slate-100
text-slate-900
bg-white

// Dark Mode (automatic)
dark:from-slate-950 dark:via-blue-950 dark:to-slate-900
dark:text-slate-100
dark:bg-slate-900
```
